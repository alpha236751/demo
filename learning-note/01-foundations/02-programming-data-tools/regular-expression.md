## 1. re 核心函数
`re` 模块最常用的6个函数，覆盖**匹配、查找、替换、分割、预编译**。

### 1.1 re.match()
**从字符串开头匹配**，仅匹配开头内容，匹配失败返回`None`
```python
# 匹配开头的"我的"
result = re.match(r'我的', text)
print(result.group())  # 输出：我的
```

### 1.2 re.search()
**全局查找第一个匹配项**，任意位置匹配，只返回第一个结果
```python
# 查找第一个"123"
result = re.search(r'123', text)
print(result.group())  # 输出：123
```

### 1.3 re.findall()
**返回所有匹配项**，以列表形式输出全部结果
```python
# 查找所有"123"
result = re.findall(r'123', text)
print(result)  # 输出：['123', '123']
```

### 1.4 re.sub()
**替换匹配内容**，语法：`re.sub(正则, 替换内容, 原字符串)`
```python
# 将所有123替换为123456
result = re.sub(r'123', '123456', text)
print(result)
# 输出：我的手机号是13812345678，备用号18998765432，固定电话010-12345678
```

### 1.5 re.split()
**按匹配内容分割字符串**，返回分割后的列表
```python
# 按"123"分割文本
result = re.split(r'123', text)
print(result)
```

### 1.6 re.compile()
**预编译正则表达式**，复用正则规则，提升效率
```python
# 预编译规则，多次使用
pattern = re.compile(r'123')
result = pattern.search(text)
print(result.group())  # 输出：123
```

## 2. 正则基本语法
正则表达式的基础匹配符号，用于匹配**单个字符**。

### 2.1 通配符 `.`
匹配**任意单个字符**（换行符除外）
```python
result = re.findall(r'.2', 'text123')
print(result)  # 输出：['12']
```

### 2.2 字符集合 `[]`
匹配括号内**任意一个字符**，支持范围、取反
```python
# 匹配1或2
result = re.findall(r'[12]', 'text123')
print(result)  # ['1','2']

# 匹配小写字母a-z
result = re.findall(r'[a-z]', 'text123')
print(result)  # ['t','e','x','t']

# 匹配非小写字母、非1-2
result = re.findall(r'[^a-z1-2]', 'text123')
print(result)  # ['3']
```

### 2.3 转义匹配符（常用简写）
| 符号 | 含义 |
| ---- | ---- |
| `\d` | 匹配任意数字（0-9） |
| `\D` | 匹配任意非数字 |
| `\w` | 匹配字母、数字、下划线 |
| `\W` | 匹配非字母/数字/下划线 |
| `\s` | 匹配空白字符（空格、制表符等） |
| `\S` | 匹配非空白字符 |

```python
# 匹配所有数字
result = re.findall(r'\d', 'text123')
print(result)  # ['1','2','3']
```

## 3. 边界匹配 & 位置匹配
用于限定匹配内容的**位置**（开头、结尾、单词边界）。

| 符号 | 含义 |
| ---- | ---- |
| `^` | 匹配**字符串开头** |
| `$` | 匹配**字符串结尾** |
| `\b` | 匹配**单词边界** |
| `\B` | 匹配**非单词边界** |

```python
# 匹配字符串开头的字符
result = re.findall(r'^\w', 'text123123')
print(result)  # ['t']

# 匹配字符串结尾的字符
result = re.findall(r'\w$', 'text123123')
print(result)  # ['3']

# 匹配单词边界后的首字符
result = re.findall(r'\b\w', 'text 123123')
print(result)  # ['t','1']

# 匹配非单词边界内容
result = re.findall(r'\B\w+', 'text 123123')
print(result)  # ['ext','23123']
```

---

## 4. 量词（控制匹配次数）
用于指定**匹配字符重复的次数**，分为贪婪/非贪婪模式。

| 符号 | 含义 | 模式 |
| ---- | ---- | ---- |
| `+` | 1次或多次（至少1次） | 贪婪 |
| `*` | 0次或多次 | 贪婪 |
| `?` | 0次或1次 | 非贪婪 |
| `{n,m}` | 重复n~m次 | 贪婪 |

```python
# 匹配1个及以上字母/数字
result = re.findall(r'\w+', 'text 123123')
print(result)  # ['text','123123']

# 匹配重复2-4次的字符
result = re.findall(r'\w{2,4}', 'text 123123')
print(result)  # ['text','1231','23']
```

## 5. 分组
用`()`包裹内容，实现**分组提取、或逻辑、分组引用**。

| 语法 | 含义 |
| ---- | ---- |
| `()` | 捕获分组，提取内容 |
| `|` | 或逻辑 |
| `(?:)` | 非捕获分组（不保存结果） |
| `\1` | 引用第一个分组内容 |

示例：
```python
# 分组提取手机号
result = re.search(r'(1\d{2})(\d{8})', text)
print(result.group(1))  # 第一个分组：手机号前3位
print(result.group(2))  # 第二个分组：手机号后8位
```

## 6. 前瞻 & 后顾（零宽断言）
不匹配字符，只**限定匹配位置**，精准筛选内容。

| 语法 | 含义 |
| ---- | ---- |
| `xxx(?=yyy)` | 前瞻：匹配**后面是yyy**的xxx |
| `xxx(?!yyy)` | 负前瞻：匹配**后面不是yyy**的xxx |
| `(?<=yyy)xxx` | 后顾：匹配**前面是yyy**的xxx |
| `(?<!yyy)xxx` | 负后顾：匹配**前面不是yyy**的xxx |
