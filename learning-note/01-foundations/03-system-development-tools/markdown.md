## 一、基本语法

### 1. 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

### 2. 引用

```markdown
> 这是一个引用
>> 这是嵌套引用（第二层）
```

> 这是一个引用  
>> 这是嵌套引用（第二层）

### 3. 列表

#### 无序列表

使用 `-`、`+` 或 `*` 均可：

```markdown
- 无序项 1
+ 无序项 2
* 无序项 3
```
- 无序项 1
+ 无序项 2
* 无序项 3

#### 有序列表

```markdown
1. 第一项
2. 第二项
3. 第三项
```
1. 第一项
2. 第二项
3. 第三项

#### 嵌套列表

```markdown
1. 有序父项
   - 无序子项
   - 另一子项
2. 有序父项第二项
```
1. 有序父项
   - 无序子项
   - 另一子项
2. 有序父项第二项

#### 任务列表（TodoList）

```markdown
- [ ] 未完成任务
- [x] 已完成任务
```
- [ ] 未完成任务
- [x] 已完成任务
  
### 4. 表格

```markdown
 左对齐 | 居中对齐 | 右对齐 
 :----- | :------: | -----: 
 1      |    2     |      3 
 4      |    5     |      6 
```

 左对齐 | 居中对齐 | 右对齐 
 :----- | :------: | -----: 
 1      |    2     |      3 
 4      |    5     |      6 

### 5. 段落与文本格式

- **换行**：行末加两个以上空格后回车，或段落之间空一行。
- **分割线**：三个及以上 `*`、`-` 或 `_` 单独成行。

```markdown
***
```
***

- **字体样式**

| 效果     | 语法           |
| -------- | -------------- |
| **加粗** | `**文本**`     |
| *斜体*   | `*文本*`       |
| ==高亮== | `==文本==`     |
| <u>下划线</u> | `<u>文本</u>` |
| ~~删除线~~ | `~~文本~~`    |

- **脚注**

```markdown
这里需要一个脚注[^1]。

[^1]: 脚注内容，通常放在文末。
```

这里需要一个脚注[^1]。

### 6. 代码

- **行内代码**：使用反引号包裹 `` `code` ``  
`print('Hello World!')`

- **代码块**：使用三个反引号包裹，可指定语言

````markdown
```python
print('Hello World!')
```
````
```python
print('Hello World!')
```

### 7. 超链接

- **行内链接**：`[链接文字](URL)`

```markdown
[DeepSeek官网](https://chat.deepseek.com/)
```
[DeepSeek官网](https://chat.deepseek.com/)

- **引用式链接**（类似脚注）

```markdown
[点击访问][deepseek]

[deepseek]: https://chat.deepseek.com/
```

[点击访问][deepseek]

### 8. 图片

- **通用格式**（推荐使用相对路径或网络URL）

```markdown
![图片描述](./images/example.png)
```

- **HTML 格式**（可控制宽高等属性）

```html
<img src="./images/example.png" alt="图片描述" width="300" />
```
