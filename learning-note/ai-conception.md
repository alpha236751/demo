# AI Agent 相关概念
AI：人工智能，让机器具备类人智能的技术总称
ML：机器学习，AI的分支，指通过数据训练模型，使机器能够自动学习和改进的技术
DL：深度学习，ML的分支，指通过多层神经网络进行学习和特征提取的技术
LLM：大型语言模型，基于深度学习的自然语言处理模型，能够理解和生成自然语言文本
Token：在自然语言处理中，指将文本分割成更小的单位（如单词、子词或字符）的过程
Context window：上下文窗口，指LLM在处理文本时，能够同时考虑的上下文信息的范围，通常以Token数量为单位
Prompt：提示词，指向LLM提供输入以引导其生成特定输出的文本
AIGC：AI生成内容，指通过AI技术自动生成文本、图像、音频等内容的过程
Transformer：一种深度学习模型架构，广泛应用于自然语言处理任务，具有自注意力机制，能够捕捉长距离依赖关系
Self-attention：自注意力机制，Transformer模型中的核心机制，使模型能够在处理输入时关注不同位置的信息
MoE：专家混合模型，指在深度学习中使用多个专家模型来处理不同类型的数据或任务，以提高模型的性能和效率
Mamba：一种新型的深度学习模型架构，旨在解决Transformer模型在处理长序列时的计算复杂度问题
KV Cache：键值缓存，指在LLM中用于存储和快速访问之前处理过的Token及其相关信息的机制，以提高模型的效率和性能
Agent：智能体，指能够感知环境、做出决策并执行任务的系统或程序
RAG：检索增强生成，指在生成内容时结合外部知识库或信息源，以提高生成结果的准确性和相关性
MCP：模型上下文协议，指在LLM中定义模型与外部工具交互的协议，让模型能够调用外部工具或API以完成特定任务
多模态：指同时处理和理解多种类型数据（如文本、图像、音频等）的能力
Fine-tuning：微调，指在预训练模型的基础上，通过少量特定任务的数据进行进一步训练，以适应特定应用场景的过程
Reinforcement Learning：强化学习，指通过与环境交互并根据反馈进行学习的机器学习方法，常用于训练智能体在复杂环境中做出决策
Function calling：函数调用，指在LLM中通过特定的输入触发预定义的函数，以实现特定的功能或任务
API：应用程序编程接口，指一组定义了软件组件之间交互的规则和协议，允许不同系统或程序之间进行通信和数据交换
SDK：软件开发工具包，指一组工具、库和文档，帮助开发者创建应用程序或系统，通常包含API和示例代码
Harness：驾驭，指AI Agent的运行外壳
Vibe Coding：感觉编程，用自然语言告诉 AI “我要什么效果”，让 AI 写代码、你只管调方向、看结果

- Pattern Recognition and Machine Learning（PRML）—Christopher Bishop
机器学习 “圣经”，理论深度天花板。
覆盖贝叶斯方法、图模型、核方法、神经网络、深度学习。
特点：数学优雅、推导严密；适合读博 / 科研进阶。
- Machine Learning: A Probabilistic Perspective（MLAPP）—Kevin Murphy
概率视角最全面教材，兼顾理论与实现。
从概率统计出发，讲解经典算法与深度学习。
特点：内容极全、代码示例丰富；适合系统构建知识体系。
- Deep Learning（花书）—Ian Goodfellow 等
深度学习奠基作，全球高校通用教材。
覆盖深度学习基础、CNN、RNN、生成模型、强化学习。
特点：理论权威、体系完整；深度学习必备案头书。
- 《机器学习》（周志华，西瓜书）
国内最权威入门 + 进阶教材，高校必修课首选。
覆盖监督 / 无监督 / 强化学习、SVM、集成学习、深度学习基础。
特点：理论严谨、案例通俗（西瓜数据集）；配套 “南瓜书” 补全公式推导。
- 《统计学习方法》（第 2 版）— 李航（微软亚洲研究院）
理论必读书，面试 / 笔试高频来源。
聚焦监督学习：感知机、K 近邻、朴素贝叶斯、决策树、SVM、EM、CRF 等。
特点：公式推导极详尽、数学味浓；适合夯实算法底层逻辑。
- 《机器学习实战：基于 Scikit-Learn、Keras 和 TensorFlow》（第 2 版）—Aurélien Géron（中文译本）
实战派 “蜥蜴书”，工业界入门首选。
全流程代码落地：数据预处理、传统模型、神经网络、CNN/RNN、强化学习。
特点：Python 代码完整、案例贴近工程；零基础友好。
